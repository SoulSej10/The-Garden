# Purpose

The Memory & History Engine preserves the past.

Its purpose is not merely to record events.

Its purpose is to determine how the past continues influencing the future.

The Garden is not only a world that changes.

It is a world that remembers.

---

# Philosophy

History is objective.

Memory is subjective.

History records everything considered historically significant.

Memory belongs to living systems.

Citizens remember differently than families.

Families remember differently than villages.

Villages remember differently than civilizations.

The distinction between history and memory creates identity.

---

# The Four Layers of the Past

The Garden preserves information across four distinct layers.

```text
Citizen Memory

↓

Family Memory

↓

Collective Memory

↓

World History
```

Each layer serves a different purpose.

---

# Layer I — Citizen Memory

Citizen Memory represents personal experience.

Examples:

People met.

Friends made.

Parents.

Children.

Places visited.

Wars survived.

Storms witnessed.

Promises made.

Traumas.

Achievements.

A citizen only remembers what they personally experienced.

Knowledge is local.

---

# Citizen Memory Characteristics

Citizen Memory:

Is subjective.

Can fade.

Can be incomplete.

Can contain emotional significance.

Can influence future decisions.

Memory should never grant omniscience.

A farmer cannot remember a distant war unless information reached them.

---

# Layer II — Family Memory

Families preserve stories beyond one lifetime.

Examples:

"Our ancestors founded this village."

"Our family survived the Great Flood."

"Our grandfather was once king."

Family Memory persists even after individuals die.

This creates legacy.

---

# Layer III — Collective Memory

Communities remember together.

Villages.

Kingdoms.

Cultures.

Religions.

Guilds.

Each develops traditions based upon shared historical experience.

Examples:

Annual harvest festivals.

Memorial days.

Taboos.

Local heroes.

Historic rivalries.

Collective Memory shapes culture.

---

# Layer IV — World History

World History is objective.

It records major events that define civilization.

Examples:

Kingdom founded.

Volcano erupted.

Great famine.

Technology discovered.

First bridge built.

Era changed.

Unlike memory, World History never forgets.

---

# Memory vs History

Memory is lived.

History is recorded.

Memory may be emotional.

History should remain factual.

Memory influences decisions.

History informs observation.

Both are essential.

---

# Event Recording

Not every Event becomes History.

Examples:

Citizen blinked.

Citizen walked.

Citizen ate lunch.

These are Events.

They are not History.

The History Engine determines historical significance.

---

# Historical Significance

An event becomes History when it changes the world in a meaningful way.

Possible criteria:

Large population impact.

Long-term consequences.

Technological breakthroughs.

Political transitions.

Environmental disasters.

Founding events.

Civilizational milestones.

The threshold may evolve over time.

---

# Memory Formation

Experiences become memories through exposure.

Example:

Citizen witnesses:

Marriage.

↓

Memory created.

↓

Citizen tells family.

↓

Family Memory strengthened.

↓

Village celebrates yearly.

↓

Tradition emerges.

↓

History references tradition.

Memory creates culture.

---

# Forgetting

Forgetting is natural.

Citizens may gradually lose insignificant memories.

Examples:

Minor conversations.

Ordinary workdays.

Routine travel.

Important experiences fade more slowly.

Trauma and celebration may persist for life.

History never forgets.

---

# Memory Strength

Every memory possesses relative strength.

Possible factors:

Recency.

Emotional impact.

Frequency of recall.

Importance.

Shared experiences.

Strong memories influence behavior more than weak ones.

---

# Cultural Transmission

Knowledge spreads.

Children inherit stories.

Teachers preserve knowledge.

Religions preserve beliefs.

Libraries preserve discoveries.

Travel spreads ideas.

Collective Memory evolves through transmission.

---

# Historical Archive

History should support exploration.

Players may inspect:

People.

Families.

Villages.

Kingdoms.

Events.

Technologies.

Disasters.

Historical timelines.

History becomes an interactive encyclopedia.

---

# Story Engine Relationship

The Story Engine consumes History.

It also consumes Memory.

Example:

History:

A king died.

Citizen Memory:

"I watched the funeral."

Family Memory:

"He protected our village."

Collective Memory:

"The Golden Age ended."

Different perspectives produce richer stories.

---

# Replay

The engine should eventually support replay through History.

Possible experiences:

Observe Year 15.

Replay the Great Migration.

Follow one family across generations.

Investigate the collapse of an empire.

History becomes explorable.

---

# Identity

Memory creates identity.

Without Memory:

Citizens become statistics.

Villages become coordinates.

Kingdoms become polygons.

With Memory:

Citizens become people.

Villages become communities.

Kingdoms become civilizations.

# Closing Statement

Chronology gives the world time.

The World State gives it form.

Rules give it behavior.

Causality gives it consequence.

Memory gives it meaning.

History gives it permanence.

Without memory, life is only motion.
