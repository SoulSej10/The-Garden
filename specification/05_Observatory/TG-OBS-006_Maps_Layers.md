# Purpose

This document defines the Observatory's mapping system.

The map is not merely a navigation tool.

It is a living atlas that allows players to visualize geography, history, ecology, civilization, and every major simulation system through layered analysis.

Every layer should help answer a different question about the world.

---

# Core Principle

**The same world should reveal different truths depending on how it is observed.**

The landscape never changes because a layer is enabled.

Only the player's understanding changes.

---

# Design Philosophy

Maps should remain geographically consistent while allowing different simulation systems to be visualized independently or in combination.

The player studies one world through many perspectives.

Every layer represents another way of understanding the same reality.

---

# Closing Statement

A player opened the map.

At first they saw only mountains, rivers, and forests.

They enabled the biodiversity layer.

The forests became habitats.

They enabled historical borders.

Ancient kingdoms appeared beneath the modern landscape.

They enabled migration routes.

Animals and people followed many of the same valleys.

Finally, they overlaid rainfall with agriculture.

The pattern became obvious.

The geography had never changed.

Only their understanding of it had.

In The Garden, every new layer reveals another way of seeing the same world.
