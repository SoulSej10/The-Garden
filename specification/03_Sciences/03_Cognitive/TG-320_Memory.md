# Purpose

Memory defines how intelligent agents preserve information across time.

Unlike perception, which exists only in the present, memory allows experiences to persist beyond the moment in which they occur.

Memory enables learning, relationships, culture, identity, planning, and history.

Without memory, intelligence cannot develop beyond immediate reaction.

---

# Scientific Basis

Memory is the process by which information is encoded, stored, strengthened, forgotten, and recalled.

Human memory is not a perfect recording of events.

It is selective, reconstructive, emotionally influenced, and constantly changing.

The Garden models memory as a living cognitive system rather than an immutable archive.

---

# Design Philosophy

The world remembers through history.

Individuals remember through memory.

The two are not always the same.

Every mind carries its own version of the past.

Those memories shape every future decision.

# Memory Types

Agents possess multiple categories of memory.

These include:

Sensory Memory

Very brief impressions lasting only moments.

Working Memory

Information currently being processed.

Short-Term Memory

Recent experiences.

Long-Term Memory

Experiences retained over extended periods.

Procedural Memory

Skills learned through repetition.

Semantic Memory

General knowledge about the world.

Episodic Memory

Personal experiences tied to specific events.

Different memory systems operate simultaneously.

---

# Memory Formation

Not every experience becomes a lasting memory.

Memory formation depends upon:

Attention.

Novelty.

Emotion.

Importance.

Repetition.

Personal relevance.

Highly significant experiences are retained more strongly.

---

# Memory Strength

Every memory possesses strength.

Strong memories are:

More durable.

More likely to be recalled.

Less vulnerable to forgetting.

Repeated recall may strengthen or alter memories.

---

# Forgetting

Forgetting is a natural cognitive process.

Memories fade because of:

Time.

Lack of use.

Interference.

Trauma.

Cognitive limitations.

Forgetting prevents unlimited memory accumulation.

---

# Memory Recall

Agents retrieve memories when relevant.

Recall may be triggered by:

People.

Places.

Objects.

Smells.

Conversations.

Emotions.

Environmental similarities.

Recall is probabilistic rather than guaranteed.

# False Memory

False memories influence behavior just as strongly as accurate ones.

# Performance Considerations

Only significant long-term memories require permanent persistence.

Routine daily experiences should decay automatically unless reinforced.

Memory compression techniques should summarize repetitive experiences while preserving important life events.

# Relationship to Civilization

Civilizations remember through people before they remember through books.

The first stories are spoken.

The first laws are remembered.

The first traditions are repeated.

Only later are they written.

A civilization with weak memory loses its identity.

A civilization with rich memory preserves knowledge across generations.

History exists because someone remembers.
